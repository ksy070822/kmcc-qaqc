## 11. CSAT 개요 (현재 구현)

### 데이터 출처

CSAT는 QA와 달리 **수동 임포트가 아닌 BigQuery 크로스 프로젝트 조회**로 실시간 집계합니다.

```
dataanalytics-25f2 (별도 GCP 프로젝트)
  ├── dw_review.review           ← 고객 평가 점수 (1~5점)
  ├── dw_review.review_request   ← 평가 요청 건
  ├── dw_cems.chat_inquire       ← 채팅 상담 건
  ├── dw_cems.consult            ← 상담 상세 (서비스 경로)
  └── dw_cems.user               ← 상담사 정보 (센터 구분)
        ↓
  lib/bigquery-csat.ts (쿼리 & 집계)
        ↓
  csat-dashboard 컴포넌트 7개
```

### 적용 범위

| 항목 | 현재 설정 | 맞나요? |
|------|-----------|:-------:|
| **채널** | 채팅 전용 (유선 제외) | [ ] ✅ / [ ] ❌ |
| **센터 구분** | team_id1: 14=광주, 15=용산 | [ ] ✅ / [ ] ❌ |
| **제외 경로** | `c2_voice_*`, `c2_cti_center_code_error` | [ ] ✅ / [ ] ❌ |
| **점수 범위** | 1~5점 (정수) | [ ] ✅ / [ ] ❌ |
