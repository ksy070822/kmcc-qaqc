# 요구사항 작성 가이드

> 각 담당자가 본인 도메인의 지표/계산식/기준값을 정리하는 폴더입니다.
> 여기에 작성된 내용을 바탕으로 메이가 대시보드를 구축합니다.

## 작성 방법

1. 본인 파일을 열어서 `[ ]` 부분을 채워주세요
2. 모르는 항목은 비워두거나 "미정"으로 적어도 됩니다
3. 작성 완료 후 메이에게 전달하면 됩니다
4. 엑셀이 더 편하면 엑셀로 작성 후 공유해도 OK

## 예시

```
지표명: 응답률
계산식: 응답건수 / 총인입건수 × 100
단위: %
목표: 용산 95%, 광주 93%
데이터 출처: CTI 시스템 → BigQuery
조회 조건: 일별, 센터별, 서비스별
```
