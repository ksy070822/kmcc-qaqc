# QC 평가 체계 도메인 지식

## 평가 항목 (16개)

### 상담태도 (5개)
| # | 항목 | 코드명 | 설명 |
|:-:|:---|:---|:---|
| 1 | 첫인사/끝인사 누락 | `greeting_error` | 인사말 미사용 |
| 2 | 공감표현 누락 | `empathy_error` | 고객 감정에 공감 없음 |
| 3 | 사과표현 누락 | `apology_error` | 불편 상황에 사과 없음 |
| 4 | 추가문의 누락 | `additional_inquiry_error` | 추가 도움 확인 안함 |
| 5 | 불친절 | `unkind_error` | 불친절한 응대 |

### 오상담/오처리 (11개)
| # | 항목 | 코드명 | 설명 |
|:-:|:---|:---|:---|
| 1 | 상담유형 오설정 | `consult_type_error` | 분류 잘못 선택 |
| 2 | 가이드 미준수 | `guide_error` | 절차 미준수 |
| 3 | 본인확인 누락 | `identity_check_error` | 본인확인 생략 |
| 4 | 필수탐색 누락 | `required_search_error` | 필수 정보 미조회 |
| 5 | 오안내 | `wrong_guide_error` | 잘못된 정보 안내 |
| 6 | 전산 처리 누락 | `process_missing_error` | 시스템 처리 누락 |
| 7 | 전산 처리 미흡 | `process_incomplete_error` | 불완전한 처리 |
| 8 | 전산 조작 오류 | `system_error` | 시스템 조작 오류 |
| 9 | ID 매핑 누락 | `id_mapping_error` | 콜/픽/트립ID 미기재 |
| 10 | 플래그/키워드 누락 | `flag_keyword_error` | 태그 미기재 |
| 11 | 상담이력 기재 미흡 | `history_error` | 상담 기록 부실 |

## 오류율 계산 (CRITICAL)
```
상담태도 오류율 = 태도오류건수 / (검수건수 x 5) x 100
오상담 오류율 = 오상담오류건수 / (검수건수 x 11) x 100
```

검수 1건에서 5개 태도항목 + 11개 오상담항목 모두 평가.
하나의 검수에서 여러 오류 발생 가능.

## 조직 구조
- **용산센터** (~200명): 택시, 퀵, 대리, 배송 (유선/채팅)
- **광주센터** (~100명): 택시, 퀵, 대리, 바이크/마스, 주차/카오너, 화물, 지금여기 (유선/채팅)

## 근속 그룹
- 신입: 0-6개월
- 주니어: 7-12개월
- 시니어: 13개월 이상
